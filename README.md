This repository provides the code and datasets for the proposed ATFN model, integrating ESG sentiment and technical indicators to predict the S&P 500 and DJIA.

The Original folder contains baseline models and datasets, while the Backtesting/Ablation folder includes code and data for validation, backtesting, and ablation studies.

## Installation
You need to install four modules or library: numpy, scipy, pandas, os, tensorflow/keras, sklearn, matplotlib
